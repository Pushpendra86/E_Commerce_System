# E_Commerce_System

Node JS Practical Task
 Build a Node.js API for an e-commerce system
 Add joi validation every filed

Users side

1. Login and registration (if new user registration, then add wallet point)
2. Product list page
3. Allow adding items to cart
4. Calculate total (consider discount seller coupons & wallet points)
5. Place an order (deduct stock + save order in DB)
6. Handle payment status (success/failure)
7. Use Express + Mysql database

Seller side

1. Create seller login and registration
2. Create seller brand and brand list
3. Seller add product and list product
4. Seller create coupon and list coupon
5. Handle product low on quantity and out of stock product

TechStack used
-->Backend
1.Node.js
2.Express.js
3.MongoDB(Mongoose ORM)

-->Authentication
1.JWT
2.bcrypt(password hashing)

--> Validation
1.JOI validation library

-->Other tools
1.dotenv
2.Postman
